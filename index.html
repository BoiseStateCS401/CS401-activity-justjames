HTML or HyperText Markup Language is one of the main components of the
World Wide Web. It allows users to create meaningful, interactive web pages
that can be dynamic and unique for different customers and businesses.

Some more resources for reading about HTML can be found at:
https://developer.mozilla.org/en-US/docs/Web/HTML
https://en.wikipedia.org/wiki/HTML

CSS or Cascading Style Sheets assist HTML pages in stylizing the look of the page
and creating engaging, interesting pages. CSS are good for pages that need to be
sytylized for the end user.

Some more resources for reading about CSS can be found at:
https://developer.mozilla.org/en-US/docs/Web/CSS
https://en.wikipedia.org/wiki/Cascading_Style_Sheets

Javascript is the last main component of the World Wide Web. It allows users to
make interactive webpages that are able to produce outcomes based upon user entered
information.
